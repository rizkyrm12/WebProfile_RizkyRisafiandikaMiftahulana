Tentang Saya

Saya merupakan mahasiswa informatika Fakultas Ilmu Komputer di Universitas Singaperbangsa Karawang. Saya senang dalam melakukan hal baru seperti mengikuti program kampus merdeka, mengikuti organisasi dan melakukan hobby sesuai passion saya.

Organisasi

Saya mengikuti organisasi di lingkungan prodi informatika yaitu HIMTIKA (Himpunan Mahasiswa Informatika UNSIKA.) Dengan mengikuti organisasi saya dapat mengasah softskill saya seperti problem solving, self management, leadership, public speaking dan lain sebagainya. Dan saat ini saya menjabat sebagai Ketua Himpunan Mahasiswa Informatika tersebut.

Hobby

Saya merupakan seseorang yang senang mnelakukan hobby sesuai passion saya salah satunya adalah bermain basket. Disana saya dapat mengasah kemampuan saya serta menyalurkan minat dan bakat saya. Saya senang melakukan hobby ini sejak dibangku SMP hingga saat ini.
